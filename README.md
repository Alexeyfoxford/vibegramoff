Мессенджер VibeGram.
Звоните,общайтесь и переписывайтесь!
двухфакторная аутентификация по почте,сквозное шифрование в ЛС и многое другое!
Ваши данные хранятся на серверах и их не поулчится никак расшифровать,так что гарантия 100% безопасности.
Регистрация в мессенджере очень простая.
Введите своё имя, пароль,почту и никнейм.
Написать человеку/создать группу,канал - кнопка "Новый".
Советы:
• Настройки → Конфиденциальность — кто может вам писать
• ПКМ по сообщению — реакции и действия
• Папки — кнопка «+» рядом с вкладками
Создать аккаунт просто: введите почту,логин,пароль и юзернейм.
